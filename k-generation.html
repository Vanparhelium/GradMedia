разработка корпоративного сайта компании, занимающейся авиационным менеджментом и консалтингом

    основная информация:

  • Вид сайта:

    Корпоративный многостраничный сайт c версией на английском языке

  • Движок:

    Интеграция с системой управления сайтом Word Press

  • Ссылка на сайт:

    https://k-generation.kz/

    Цели:

  • подача информации о компании для потенциальных клиентов и партнеров

  • поддержание имиджа компании в Сети

    выполненные работы:

  1. Разработка структуры сайта

    Проведение анализа деятельности компании, распределение информации по базовым страницам, привычным для навигации любому современному пользователю

  2. информационная подача

    Написание текстов профессиональным копирайтером. Информация подана кратко, доступно, с учетом необходимых фактов о деятельности компании.


    При написании текстов копирайтер стремился избегать “перегруза” сайта излишними текстами.

  3. визуальное оформление

    Дизайн страниц сайта строгий и одновременно стильный, соответствующий направлению деятельности компании.


    В дизайне использованы профессиональные фото сотрудников, офиса, борта самолета компании, которые наиболее ярко расcказывают о сути деятельности компании.

  4. верстка сайта

    Готовый дизайн сайта сверстан программистом на HTML и CSS коде с адаптацией под мобильные телефоны и планшеты.

  5. Интеграция с word press

    Завершающим этапом была интеграция сайта с системой управления Word Press, благодаря чему владелец сайта в дальнейшем может самостоятельность обновлять основные разделы и добавлять новости компании самостоятельно.
